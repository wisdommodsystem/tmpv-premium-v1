# 🤖 WISDOM TEMP Bot - دليل التشغيل 24/7

## 🚀 طرق تشغيل البوت بشكل مستمر

### الطريقة الأولى: استخدام ملف Batch (الأسهل)
```bash
# انقر مرتين على الملف
start_bot.bat
```

### الطريقة الثانية: استخدام PowerShell
```powershell
# افتح PowerShell كمدير وشغل الأمر
.\start_bot.ps1
```

### الطريقة الثالثة: التشغيل العادي (مع الحماية المدمجة)
```bash
node index.js
```

## 🛡️ ميزات الحماية المضافة

### 1. معالجة الأخطاء التلقائية
- **Uncaught Exceptions**: البوت لن ينطفئ عند حدوث خطأ غير متوقع
- **Unhandled Rejections**: معالجة الوعود المرفوضة تلقائياً
- **Discord Errors**: إعادة الاتصال التلقائي عند انقطاع الاتصال

### 2. نظام إعادة الاتصال
- إعادة الاتصال التلقائي عند انقطاع الإنترنت
- إعادة المحاولة كل 5 ثوانٍ في حالة فشل تسجيل الدخول
- رسائل واضحة لحالة الاتصال

### 3. مراقبة الحالة
- عرض عدد الخوادم والمستخدمين
- حالة البوت تظهر "Voice Channels | .v help"
- سجلات مفصلة لجميع الأحداث

## � نظام الـBlacklist (منع إنشاء القنوات)

- تخزين بسيط في `data/blacklist.json` مع صلاحيات انتهاء زمنية.
- أوامر `.v blacklist @user <time> <reason>` و `.v rblacklist <user>` (Admins only).
- يتم منع المستخدم من الانضمام إلى قناة "Create Room" إذا كانوا في القائمة.
- أي مدخلات غير صالحة تُتجاهل، ويتم فصل المستخدم فوراً إذا كانوا داخل "Create Room".

## 🔧 استكشاف الأخطاء

### إذا كان البوت لا يزال ينطفئ:
1. تأكد من صحة التوكن في ملف `.env`
2. تحقق من اتصال الإنترنت
3. تأكد من أن البوت مفعل في Discord Developer Portal
4. استخدم `start_bot.bat` للإعادة التلقائية

### إذا كانت هناك مشاكل في الأذونات:
1. تأكد من أن البوت لديه الأذونات المطلوبة
2. تحقق من أن البوت في الخادم
3. تأكد من أن `ADMIN_ROLE_ID` صحيح

## 💡 نصائح للتشغيل المستمر

1. **استخدم VPS أو خادم مخصص** للتشغيل 24/7 الحقيقي
2. **استخدم PM2** لإدارة العمليات بشكل أفضل:
   ```bash
   npm install -g pm2
   pm2 start index.js --name "wisdom-bot"
   pm2 startup
   pm2 save
   ```
3. **راقب استهلاك الذاكرة** بانتظام
4. **احتفظ بنسخة احتياطية** من ملفات البوت

## 🌐 للتشغيل على خادم

### استخدام PM2 (الأفضل للخوادم):
```bash
# تثبيت PM2
npm install -g pm2

# تشغيل البوت
pm2 start index.js --name "wisdom-temp-bot"

# حفظ التكوين
pm2 startup
pm2 save

# مراقبة البوت
pm2 monit
```

### استخدام Screen (Linux):
```bash
# إنشاء جلسة جديدة
screen -S wisdom-bot

# تشغيل البوت
node index.js

# الخروج من الجلسة (Ctrl+A ثم D)
# العودة للجلسة
screen -r wisdom-bot
```
